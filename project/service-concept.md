# Lexora — サービスコンセプト

## 名前の由来

**Lex**（ラテン語: 言葉・法則）+ **Aurora**（夜明け）= **Lexora**

「言葉の夜明け」——非英語話者の研究者が、言語の壁を越えて研究成果を世界に届けるための翻訳パートナー。

## ミッション

> 言語が科学を制限してはならない。

世界中の研究者が、母語で思考し、任意の言語で発信できる未来を実現する。

## ブランドトーン

| 属性 | 説明 |
|------|------|
| **知的** | 学術コミュニティに相応しい品格 |
| **静か** | 派手さより信頼感。落ち着いたトーン |
| **誠実** | 翻訳品質とコストに対する透明性 |
| **力強い** | 研究者の背中を押す確かなサポート |

## ビジュアルアイデンティティ

- **ロゴ書体**: Merriweather (serif) — 学術的な信頼感
- **本文書体**: Outfit (sans-serif) — 現代的な可読性
- **配色**: 既存テーマを踏襲（secondary bg + foreground テキスト）
- **トーン**: ダークセクション (`bg-secondary`) とライトセクション (`bg-background`) の交互配置

## ターゲットユーザー

- 英語での論文執筆に時間がかかる非英語話者の研究者
- 国際ジャーナルへの投稿を目指す大学院生・ポスドク
- 多言語チームでの共同研究者

## 差別化ポイント

1. **学術特化** — 汎用翻訳ではなく、ジャーナル別スタイルに対応
2. **双方向リアルタイム** — 原文と翻訳を同時に編集・同期
3. **LaTeX対応** — `\cite{}`, `\ref{}` 等のLaTeXコマンドを翻訳時に保持・適切に配置
4. **コスト透明性** — リアルタイムでAPIコストを表示
5. **論文構成チェック** — AIによるIMRAD構成の自動検証
