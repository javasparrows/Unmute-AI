# プロジェクト概要 — Unmute AI

研究者が論文を効率的に書くためのAIプラットフォーム。引用管理、エビデンス探索、多言語執筆、ジャーナルスタイル対応、構成チェックを統合したエディタを提供する。

## 背景

- 論文執筆のツールが断片化しており、エディタ・翻訳・引用管理・構成チェックが別々で非効率
- 適切な引用の探索・挿入に膨大な手作業が発生している
- 非英語話者は英語話者と比べて論文執筆に50%多くの時間を費やしている
- ジャーナルごとに異なるスタイル要件への対応が煩雑

## 技術スタック

- Next.js (App Router)
- Gemini API (gemini-2.5-flash-lite)
- Vercel にデプロイ (Git push で自動デプロイ)
- Vercel Postgres (Neon) — データベース
