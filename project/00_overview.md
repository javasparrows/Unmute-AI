# プロジェクト概要 — Unmute AI

英語が苦手な日本人研究者がスムーズに英語論文を書けるリアルタイム双方向翻訳エディタ。

左側に元の文章、右側に翻訳後の文章を表示し、どちらを修正してもリアルタイムで対応する側が更新される。Gemini APIを使用した翻訳、言語自動検出、論理構成チェック機能を備える。

## 背景

- ChatGPT等を使った翻訳ワークフローはcopy & pasteが頻繁に発生し非効率
- リアルタイムで翻訳を見ながら日本語・英語の修正を行き来したいというニーズ

## 技術スタック

- Next.js (App Router)
- Gemini API (gemini-2.5-flash-lite)
- Vercel にデプロイ (Git push で自動デプロイ)
- Vercel Postgres (Neon) — データベース
