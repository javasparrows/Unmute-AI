# フロントエンド設計

## レイアウト

- ヘッダー: タイトル、翻訳ステータス、コスト表示、構成チェック・履歴・設定ボタン
- 言語バー: 左言語セレクター、入れ替えボタン、右言語セレクター、ジャーナルセレクター、プロバイダセレクター、クリアボタン
- エディタ領域: 左パネル | 同期ボタン列 | 右パネル の3列構成

### 同期ボタン列

パネル間の中央に配置される縦並びのボタン列:
- 「→」ボタン: 原文の変更を翻訳に反映（`ArrowRight` アイコン）
- 「←」ボタン: 翻訳の変更を原文に反映（`ArrowLeft` アイコン）
- 翻訳中はボタンに `Loader2` アイコン（回転アニメーション）を表示
- 各ボタンにTooltipで操作説明を表示
- テキストが空の場合はボタン無効化

## エディタパネル

- TipTap エディタ（リッチテキストエディタ）
- 文単位ハイライト対応
- IME対応: `onCompositionStart` / `onCompositionEnd` で日本語入力を制御

## カーソル連動（アラインメントベース文ハイライト）

### 従来の方式（廃止）

位置ベースの1:1マッピング（原文のindex N = 翻訳のindex N）。
翻訳で文が統合・分割されるとズレが発生する。

### 新方式

翻訳後にLLMアラインメントAPIでソース↔ターゲットのN:M対応マップを取得。
クリック時にマップを逆引きして、対応する文（複数可）をハイライトする。

### 状態管理

```typescript
// 従来: 単一インデックスを両パネルで共有
activeSentenceIndex: number | null;

// 新: パネルごとに複数インデックス
interface HighlightState {
  leftIndices: number[];
  rightIndices: number[];
}
```

### ハイライト変換フロー

```
左パネルでクリック (sourceIndex = 2)
  → alignmentMap.sourceToTarget.get(2) = [1]
  → 左パネル: [2] をハイライト
  → 右パネル: [1] をハイライト

右パネルでクリック (targetIndex = 1)
  → alignmentMap.targetToSource.get(1) = [1, 2]
  → 右パネル: [1] をハイライト
  → 左パネル: [1, 2] をハイライト (統合された2文をまとめてハイライト)
```

### フォールバック

アラインメント未取得時（取得中、エラー時）は従来の位置ベース1:1マッピングにフォールバック。

### スクロール位置同期

- 既存の `useScrollSync` はそのまま維持

## コスト表示 (ヘッダー)

ヘッダーの TranslationStatus の右に Separator を挟んで配置。

- **DeepL**: Progress バー (w-20, h-1.5) + パーセンテージ表示
  - 無料枠: 500,000文字/月
  - 計算: `(characters / 500_000) * 100`
  - Tooltip: 現在の文字数 / 上限、月次リセット説明
- **Gemini**: 推定コスト (USD) をテキスト表示
  - Input: $0.075 / 1M tokens
  - Output: $0.30 / 1M tokens
  - 計算: `(inputTokens / 1M) * 0.075 + (outputTokens / 1M) * 0.30`
  - Tooltip: トークン数内訳、レート表示
- データは localStorage (`translation-costs`) に永続化、月次自動リセット

## プロバイダセレクター (言語バー)

JournalSelector の右に Separator を挟んで配置。

- shadcn/ui Select コンポーネント (w-[130px], h-8)
- 選択肢: DeepL / Gemini
- localStorage (`translation-provider`) に永続化
- 切り替え時: 即座の再翻訳は行わない（次の同期ボタン押下時に新プロバイダで翻訳）

## デザイン

### AstroVista カラーテーマ

OKLCH色空間を使用したカスタムカラーテーマ。

#### Light モード
| CSS変数 | 値 | 用途 |
|---|---|---|
| `--background` | `oklch(0.98 0.005 280)` | 背景 |
| `--foreground` | `oklch(0.15 0.02 280)` | テキスト |
| `--primary` | `oklch(0.55 0.18 280)` | アクセント |
| `--card` | `oklch(0.99 0.003 280)` | カード背景 |
| `--muted` | `oklch(0.94 0.01 280)` | ミュート背景 |

#### Dark モード
| CSS変数 | 値 | 用途 |
|---|---|---|
| `--background` | `oklch(0.13 0.02 280)` | 背景 |
| `--foreground` | `oklch(0.93 0.01 280)` | テキスト |
