# 技術スタック

## フロントエンド / フレームワーク

- **Next.js** (App Router) — `writing_paper_other_language/` ディレクトリ内に構築
- **TypeScript**
- **Tailwind CSS** + **shadcn/ui**

## AI / LLM

- **Gemini API** (gemini-2.5-flash-lite)
- `GEMINI_API_KEY` を環境変数からロード
- Vercel AI SDK (`ai` + `@ai-sdk/google`) を使用
